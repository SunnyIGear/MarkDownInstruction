#Подсказки по командной строке

Команда смены дериктории
```sh
cd c:\folder_name
```

Команда отображения текущей директории
```sh
dir
```

Листинг текущей директории
Windows:
```sh
dir
```
Linux, MacOS
```sh
pwd
```

Удаление файлв в Windows:
```sh
del <filename>
```

Удаление файла в Linux, MacOS:
```sh
rm <filename>
```
